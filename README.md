# Ataque Brute Force com Medusa
Neste projeto, foi realizada uma simulação de um ataque de força bruta usando o Medusa. Além disso, foram utilizadas diversas ferramentas para obter as informações necessárias para a concretização do ato. É imprescindível destacar que este trabalho foi efetuado para fins educacionais apenas, ou seja, feito em um ambiente controlado através da virtualização. Abaixo está listado as principais funções usadas.

### Sistema:
- Backbox Linux: Esta é uma distribuição Linux feita para arquitetar e concretizar Pen Test, porém, de forma minimalista e mais enxuta. Usado por causa da sua compatibilidade e uso de recursos controlado.
- Metaspoitable 2: É um sistema vulnerável útil para simular estes ataques direcionados.

### Ferramentas:
- fping: Função para escanear múltiplos pings simultaneamente - averiguando a rede como um todo.
- nbtscan: Ferramenta que realiza o escaneamento para averiguar detalhes mais específicos sobre os dispositivos conectados à rede.
- nmap: Utilizado para ilustrar serviços e portas que podem estar abertas no determinado host atingido.
- medusa: Ferramenta principal para a efetuação certeira do ataque força bruta, usando uma wordlist ela força ataques direcionados a um alvo informado.
